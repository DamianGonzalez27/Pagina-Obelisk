MySQL

Objetivo del curso

El software MySQL proporciona un servidor de base de datos SQL muy rapido, multiusuario y robusto.


Esta diseñado para entornos de produccion criticos con alta carga de trabajo asi como para integrarse en software para ser distribuido.


Los usuarios pueden elegir entre usar el software MySQL como un producto Open Source bajo los terminos de la licencia GNU, o pueden adquirir una licencia comercial estandar.


Este curso esta diseñado para que los estudiantes puedan implementar las bases de datos.

Que tengan un amplio panorama de los alcances de MySQl y asi poder implementarlo a su conveniencia.
